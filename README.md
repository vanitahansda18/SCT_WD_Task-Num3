# 📝 To-Do List Web App

A simple and attractive **To-Do List Web Application** built with **HTML, CSS, and JavaScript**.
This app helps users manage their daily tasks by adding tasks with **date, time, and alarm reminders**.

## 🌐 Live Demo

👉 https://thunderous-salamander-6b831d.netlify.app/

## ✨ Features

✔ Add new tasks
✔ Set **date and time** for tasks
✔ Alarm / reminder for tasks
✔ Simple and clean UI
✔ Responsive design

## 🛠️ Built With

* HTML
* CSS
* JavaScript

## 🎯 Learning Purpose

This project was created to practice **front-end development** and improve skills in **JavaScript DOM manipulation and UI design**.

## 🚀 Deployment

The project is deployed using **Netlify**.
